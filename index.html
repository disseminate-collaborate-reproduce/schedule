Before the Workshop

  • Make sure to have created a GitHub account and report your username in this short survey .

  • A few days after filling out the survey, you will receive an invitation to join our GitHub organization, please accept this invitation.

  • Install R Yes, you do need to download and install R even if you have downloaded before. There is a newer version.

  • Install RStudio Yes, you do need to download and install RStudio even if you have downloaded before. There is a newer version. Download the free Desktop version.

  • Install and setup git

  • Test your installations . If you have filled out the survey later than usual, your test repo may not be available. Please contact us in that case.
